Exam-Style Questions

Q1: What is a pointer?

Answer: an object that stores an address

Q2: Declare a pointer that points to eecs.

Answer: int * ptr = &eecs;

Q3: Using only the pointer that you just created, change the value of eecs to 280.

Answer: *ptr = 280;

Q4: Arrays have a fixed size that must be known at compile time.

Answer: True!

Q5: Elements of a vector are contiguous in memory.

Answer: True!

Q6: What's happening on line 3?

Answer: You're doing an assignment!

Q7: What's happening on line 2?

Answer: You're making a copy!

Q8: When you create a reference you create a new object.

Answer: False!

Q9: Unit testing is when you test an entire project.

Answer: False!

Q10: Recall from project 1 the functions inside of stats.cpp. Procedural Abstraction means that if you changed the implementation of any of the functions, the code that calls those functions does not need to change.

Answer: True!

Q11: Which of the following are properties of Procedural Abstraction?

  • local
  • compiled
  • declarative
  • procedural
  • substitutable

Answer: local and substitutable!

New Vocab

  • Struct: The struct definition creates a new data type: (which is composed of subobjects). A struct only has data members and they are public by default.

  • Class: The class definition creates a new, unique data type: (which is composed of subobjects). A class can have both member variables and member functions -- they are by default private.

  • Abstract Data Type (ADT): a model of a certain kind of data structure. Focuses on what the structure does – not how it does it (an abstraction).

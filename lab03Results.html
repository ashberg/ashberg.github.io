Exam-Style Questions

Q1: There is a null character at the end of every c-style string.

Answer: True!

Q2: Given the command line input below, what is argc?

Answer: 7

Q3: Given the command line input below, where's Waldo located?

Answer: argv[5]

Q4: What does "cout << *argv[2] << endl;" print?

Answer: S

Q5: IO Objects MUST be passed by reference.

Answer: True!

Q6: Use only tvPtr to change the value of tvShow to be your current favorite tv show.

Answer: *tvPtr = "The Office";

Q7: Without using tvShow explicitly, create another pointer that will hold the address of tvShow. (Hint: use tvPtr)

Answer: string *anotherPtr = tvPtr;

Q8: Create a pointer to tvPtr called ptr2ptr.

Answer: string **ptr2ptr = &tvPtr;

Q9: Use only ptr2ptr to change the value of tvShow to be a tv show that you liked when you were little.

Answer: **ptr2ptr = "The Wild Thornberrys";

Q11: Which pointer would you want to use if you wanted to make sure that the pointer would always point at "thing" but could modify "thing"?

  • const int * ptr = &thing;
  • int * const ptr = &thing;
  • const int * const ptr = &thing;
  • int *ptr = &thing;

Answer: int * const ptr = &thing;

Q12: Which pointer would you want to use if you wanted to make sure that "thing" is never modified, but that the pointer can point elsewhere?

  • const int * ptr = &thing;
  • int * const ptr = &thing;
  • const int * const ptr = &thing;
  • int *ptr = &thing;

Answer: const int * ptr = &thing;
